# 📄 HTML Activities Project

## 📌 Overview

This project is a collection of simple HTML activities that demonstrate basic web development skills such as creating tables, buttons, and forms. It focuses on structuring content and adding simple interactivity.


## ✨ Features

* **Class Schedule Table** – shows a weekly schedule using a table with merged cells (break & free time)
* **Interactive Buttons** – includes:
  - Alert button (shows a message)
  - Redirect button (opens a website)
  - Reset button
* **Student Registration Form** – includes:
  - Full Name, Email, Password
  - Gender (radio buttons)
  - Hobbies (checkboxes)
  - Course (dropdown)
  - Message (textarea)
  - Submit & Reset buttons

## 🎨 Styling

Basic styling is applied using inline CSS:

* Clean table borders using `border-collapse`
* Spacing and padding for readability
* Centered and bold text for important rows (e.g., Break)
* Organized and readable form layout
